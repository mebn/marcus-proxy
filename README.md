# Marcus Proxy

A forward proxy written in Go for capturing and inspecting mobile app API traffic.

## Development

Start a development server with hot reload:

```bash
wails dev
```

### Prerequisites

- [Go](https://go.dev/)
- [Wails](https://wails.io/)
